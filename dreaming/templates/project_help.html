Что это

AI Dreaming Center — оркестратор Claude CLI сессий, разнесённый по проектам. Каждая строка в projects — это отдельная micode-папка со своими агентами, настройками и расписанием ночного обучения.

Где живут агенты этого проекта

В {{ project.working_dir }}/.claude/agents/ — оттуда тянутся определения. Sidebar слева переключает между разделами этого проекта; проектный select сверху — между проектами.

Полезное

  • • Глобальные дефолты живут на /settings — per-project overrides на этой странице.
  • • Сессии Claude CLI запускаются ProcessManager-ом; их stdout/stderr пишется в БД и стримится через /p/{{ project.slug }}/live.
  • • Расписание (ночное обучение) держит AsyncIOScheduler с per-project job'ами.
  • • Если что-то выглядит криво — сначала проверь рестарт uvicorn: --reload не подтягивает i18n кэш на лету (он читается один раз на старте I18n).
